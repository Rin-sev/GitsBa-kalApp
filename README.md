# UPV KomHack 2026 — RiskReady

## App Name: **kalApp**

**kalApp** (kalap) is a native Android application designed for Local Government Units (LGUs) in the Philippines to monitor and coordinate citizen welfare during disasters, primarily typhoons and other calamities.

The app operates at the **barangay level** — the smallest administrative unit in the Philippines — enabling **household-level status reporting** to a designated barangay coordinator.

Because it uses **SMS over 2G networks**, kalApp remains functional even when mobile data is unavailable or congested — a common scenario during large-scale disasters.

---

## Key Features

- **Three-status reporting**  
  `URGENT`, `EVACUATED`, `SAFE`

- **Household-level identification**  
  Each report is tagged with a barangay household ID

- **Live coordinator dashboard**  
  Incoming log feed with per-status counters:  
  `UNKNOWN`, `URGENT`, `EVACUATED`, `SAFE`, `DISPATCHED`

- **Acknowledge system**  
  Coordinators can mark reports as responded to

- **2G SMS transport**  
  Works without mobile data; resilient to network congestion

- **Offline-first architecture**  
  No backend server, no Firebase, no cloud dependency
